# Data Structures & Algorithms (DSA) 🚀

This repository is a collection of my solutions to various Data Structures and Algorithms problems. I am currently following the **Striver A2Z DSA Course/Sheet** to build a solid foundation in problem-solving and algorithmic thinking.

## 📌 Overview

The goal of this repository is to track my progress, document efficient ways to solve problems, and maintain a clean codebase for future reference. All solutions are implemented in **C++**.

## 📂 Repository Structure

The repository is organized into categories based on problem types and sources:

- **`Patterns/`**: Implementation of various star and number patterns to master nested loops.
- **`Recursion/`**: Problems covering basic recursion to advanced backtracking.
- **`Sorting/`**: Implementations of Bubble, Selection, Insertion, Merge, and Quick Sort.
- **`Array_Questions/`**: Comprehensive list of array-based challenges (Easy to Hard).
- **`Hashing/`**: Solutions involving frequency counting and hash maps.
- **`leetcode/`**: Specific folders named by LeetCode problem IDs for easy lookup.
- **`CodeForces/`**: Solutions to competitive programming problems from CodeForces contests.

## 🛠️ Tech Stack

- **Language:** C++17 or higher
- **Compiler:** GCC / Clang
